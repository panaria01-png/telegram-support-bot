# Telegram Support Bot (3 отдела + автоочередь)

## Как работает
- Клиент пишет боту в ЛС
- Выбирает тему: Продажи / Поддержка / Доставка
- Бот создаёт заявку №...
- Бот назначает ответственного автоматически (очередь внутри отдела)
- В группу отдела уходит карточка заявки
- Операторы отвечают Reply на карточку — бот отправляет ответ клиенту
- У клиента всегда 1 активная заявка
- История хранится в SQLite

## Установка (Windows)
1) python -m venv .venv
2) .\.venv\Scripts\Activate.ps1
3) pip install -r requirements.txt
4) copy .env.example .env и заполнить
5) python main.py

## Команды
- /chat_id (в группе) — узнать chat_id
- /find <ticket_no|@username|client_id> — поиск
